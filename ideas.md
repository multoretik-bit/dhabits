# dHabits Design Ideas

## Approach 1: Dark Cyberpunk Productivity
**Design Movement:** Cyberpunk / Dark Neon
**Core Principles:** Dark backgrounds with neon accents, sharp edges, monospace data readouts
**Color Philosophy:** Deep navy/black base with electric indigo (#6366f1) accent, neon green for success states
**Layout Paradigm:** Sidebar navigation + content area, data-dense cards
**Signature Elements:** Neon border glows, coin counter as HUD element, progress bars with gradient fills
**Interaction Philosophy:** Snappy, immediate feedback with subtle glow effects on hover
**Animation:** Fast transitions, number counting animations for coins
**Typography:** JetBrains Mono for data, Inter for UI

## Approach 2: Warm Minimal Productivity (CHOSEN)
**Design Movement:** Warm Minimalism / Japanese Wabi-sabi
**Core Principles:** Warm dark tones, intentional whitespace, subtle depth, organic feel
**Color Philosophy:** Deep warm charcoal (#1a1a1a) background, warm amber/gold accent (#f59e0b), soft cream text
**Layout Paradigm:** Compact sidebar + main content, card-based habit display with left color borders
**Signature Elements:** Coin display as golden badge, colored left-border habit cards, streak flames
**Interaction Philosophy:** Smooth transitions, satisfying completion animations
**Animation:** Gentle fade-ins, progress bar fills, button press effects
**Typography:** Poppins for headings, system-ui for body

## Approach 3: Glassmorphism Dark
**Design Movement:** Glassmorphism / Frosted Glass
**Core Principles:** Semi-transparent cards, blur effects, layered depth
**Color Philosophy:** Deep purple/blue gradient background, glass cards with white borders
**Layout Paradigm:** Full-width layout with floating glass panels
**Signature Elements:** Frosted glass cards, gradient backgrounds, backdrop blur
**Interaction Philosophy:** Smooth hover lifts, glass shimmer effects
**Animation:** Blur transitions, float animations
**Typography:** Inter + Space Grotesk
