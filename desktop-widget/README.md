# dHabits Activity Widget

Маленький always-on-top виджет для Windows, который считает, сколько времени вы
проводите в 4 режимах: **Работа**, **Учёба**, **Спорт**, **Бесполезное**.

Работает полностью локально — не подключается к вашему аккаунту dHabits и не
отправляет ничего в сеть. Данные хранятся рядом со скриптом, в файле
`activity_data.json`.

## Установка и запуск

```powershell
cd desktop-widget
pip install -r requirements.txt
python dhabits_widget.py
```

Появится узкое окошко у правого края экрана. По умолчанию активен режим
«Бесполезное» — переключайтесь на нужный режим кнопкой, когда садитесь
работать/учиться/заниматься спортом, и обратно, когда закончите.

## Автозапуск при включении ПК

```powershell
cd desktop-widget
.\install_autostart.ps1
```

Создаст ярлык в папке автозагрузки Windows, который запускает виджет без
консольного окна (через `pythonw.exe`).

## Уведомления

Если вы провели **2 часа подряд** в режиме «Бесполезное», придёт уведомление
Windows с предложением переключиться. Между **00:00 и 10:00** уведомления не
присылаются (время сна). Переключение в любой другой режим сбрасывает таймер
до следующего уведомления.

## Кнопка «Показать активность за день»

Показывает, сколько минут сегодня набежало в каждом из 4 режимов.

## Данные

Всё хранится в `activity_data.json` рядом со скриптом:

```json
{
  "2026-07-17": { "work": 42.3, "study": 0, "sport": 15.0, "idle": 96.7 }
}
```
